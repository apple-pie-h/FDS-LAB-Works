
# Foundation of Data Science (FDS) – Lab Work

This repository contains the **lab work and practical exercises** for the subject **Foundation of Data Science (FDS)**, taught in the **3rd Semester of BCT (Bachelor in Computer Engineering)** under **IOE (Institute of Engineering)**.

## 📌 Course Details
- **Subject:** Foundation of Data Science (FDS)
- **Semester:** 3rd
- **Program:** BCT (Bachelor in Computer Engineering)
- **University:** Institute of Engineering (IOE)

## 📂 Repository Contents
This repository includes:
- Lab exercises and assignments
- Python programs related to data science fundamentals
- Data preprocessing and analysis tasks
- Data visualization and basic machine learning implementations (if applicable)
- Well-structured and commented code for learning and reference

## 🛠 Tools & Technologies Used
- **Google Colab**
- Python
- NumPy
- Pandas
- Matplotlib / Seaborn (if applicable)

The main objectives of this repository are:
- To practice fundamental concepts of data science
- To maintain organized lab work for academic reference
